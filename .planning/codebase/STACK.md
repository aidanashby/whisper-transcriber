# Technology Stack

**Analysis Date:** 2026-04-30

## Languages

**Primary:**
- Python 3.11+ — all application code, build tooling

## Runtime

**Environment:**
- Windows desktop (win32); app targets Windows users exclusively
- No server component; fully local execution

**Package Manager:**
- pip
- Lockfile: not present (requirements.txt pins minimum versions only)

## Frameworks

**UI:**
- customtkinter 5.2.2 — themed tkinter widget set for the desktop GUI
- tkinterdnd2 0.4.3 — native Windows drag-and-drop support over tkinterdnd2.TkinterDnD.Tk root

**ML Inference:**
- faster-whisper 1.2.1 — CTranslate2-backed Whisper inference (WhisperModel)
- ctranslate2 4.7.1 — underlying inference engine; GPU path uses float16, CPU path uses int8

**Build:**
- PyInstaller 6.x — packages app as `--onedir` bundle (`transcriber.spec`)
- NSIS (optional) — produces `WhisperTranscriber-Setup.exe` installer; falls back to ZIP if absent

## Key Dependencies

**Critical:**
- `faster-whisper>=1.0.0` — model loading and transcription; hardcoded to `large-v3` model
- `customtkinter>=5.2.0` — all visible UI widgets
- `tkinterdnd2>=0.3.0` — drag-and-drop; must be root window type
- `imageio-ffmpeg>=0.4.9` — bundles static ffmpeg binary used for audio pre-processing
- `huggingface_hub>=0.20.0` — model download from Hugging Face on first run
- `pillow>=10.0.0` — icon generation (`create_icon.py`)

**GPU acceleration (Windows-specific):**
- `nvidia-cublas-cu12>=12.3` — supplies `cublas64_12.dll`, `cublasLt64_12.dll`
- `nvidia-cuda-runtime-cu12>=12.3` — supplies `cudart64_12.dll`
- These DLLs are bundled explicitly in `transcriber.spec:_find_nvidia_dlls()` because ctranslate2 does not bundle them

**Tokenisation (hidden imports):**
- `tokenizers 0.22.2` — Whisper tokeniser
- `pydantic` — used internally by faster-whisper; missed by PyInstaller auto-analysis

## Configuration

**Environment:**
- No `.env` file; no runtime environment variables required by the app itself
- Model cache stored at `%LOCALAPPDATA%\WhisperTranscriber\models\` (falls back to `~`)
- Log file at `%LOCALAPPDATA%\WhisperTranscriber\transcription.log`

**Build:**
- `transcriber.spec` — PyInstaller spec; entry point is `run.py`
- `build.bat` — one-command build script: installs deps, generates icon, runs PyInstaller, packages output
- `assets/icon.ico` — generated by `create_icon.py` using Pillow

## Platform Requirements

**Development:**
- Python 3.11+
- pip
- NSIS (optional, for installer builds)

**Production:**
- Windows only (tkinterdnd2 Tcl extension, CUDA DLL paths, `CREATE_NO_WINDOW` subprocess flag)
- NVIDIA GPU optional; CPU-only mode supported transparently
